# Pomodoro Timer

黒背景にネオンカラーを使ったモダンなデザインのポモドーロタイマーです。
ブラウザさえあれば、インストール不要でどのOSでも動作します。

---

## ポモドーロ法とは？

「25分集中 → 5分休憩」を1セットとする時間管理術です。
短いサイクルで区切ることで、集中力を維持しやすくなります。

---

## 機能

| 機能 | 内容 |
|------|------|
| 3つのタイマーモード | Pomodoro（25分）/ Short Break（5分）/ Long Break（15分） |
| リングプログレスバー | 残り時間を円弧でビジュアル表示 |
| START / STOP / RESET | タイマーの開始・一時停止・リセット |
| セッションドット | ポモドーロを完了するたびに点灯（4セット管理） |
| 終了通知 | タイマー終了時にビープ音 ＋ タブタイトルが点滅 |
| モード別カラー | Pomodoro: シアン / Short Break: グリーン / Long Break: パープル |

## 使い方

### 1. ファイルを開く

クローン（またはダウンロード）したフォルダの中の以下のファイルをダブルクリックします。

```
3.CPO/Pomodoro/index.html
```

ブラウザが自動的に開き、タイマーが表示されます。

### 2. モードを選ぶ

画面上部のタブでモードを切り替えます。

| タブ | 時間 | 使うタイミング |
|------|------|----------------|
| Pomodoro | 25分 | 集中して作業するとき |
| Short Break | 5分 | ポモドーロ1セット後の短い休憩 |
| Long Break | 15分 | 4セット完了後の長い休憩 |

### 3. タイマーを操作する

| ボタン | 動作 |
|--------|------|
| START | タイマーを開始する |
| STOP | タイマーを一時停止する（再度STARTで再開） |
| RESET | タイマーをそのモードの最初に戻す |

### 4. セッション管理

画面下部の4つのドットがセッション数を示しています。
ポモドーロを1回完了するたびに1つ点灯し、4セット完了でリセットされます。

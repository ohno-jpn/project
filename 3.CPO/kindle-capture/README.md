# Kindle Capture

Windows上のKindleアプリを自動操作し、全ページをスクリーンショットして1つのPDFにまとめるツールです。
GUIで作者名・タイトル・ページ送り方向・ページ数を入力するだけで、自動的にキャプチャが始まります。

> **注意：Windows専用です。** macOSでは動作しません。

---

## 機能

| 機能 | 内容 |
|------|------|
| GUIで設定入力 | 作者・タイトル・ページ送り方向・ページ数をウィンドウで入力 |
| 自動キャプチャ | KindleをフルスクリーンにしてページをPNG画像で自動保存 |
| 自動終了検出 | ページ数未指定の場合、ページが変わらなくなったら自動終了 |
| PDF生成 | キャプチャした画像を1つのPDFファイルに結合 |
| 後処理 | 一時ファイルの削除確認・PDFの最終フォルダへの移動 |
| 完了通知 | 処理完了時にビープ音で通知 |

---

## 動作環境

| 項目 | 要件 |
|------|------|
| OS | **Windows 10 / 11**（必須） |
| Python | 3.10 以上 |
| Kindleアプリ | Windows版 Kindle（事前に起動しておく） |

## インストール方法

### 1. リポジトリをクローンする

```bash
git clone https://github.com/ohno-jpn/2026a.git
cd 2026a/3.CPO/kindle-capture
```

### 2. 仮想環境を作成する（推奨）

仮想環境とは、このプロジェクト専用のPython環境のことです。
他のプロジェクトに影響を与えずにライブラリをインストールできます。

```bash
python -m venv .venv
.venv\Scripts\activate
```

> ターミナルの先頭に `(.venv)` と表示されれば成功です。

### 3. 必要なライブラリをインストールする

```bash
pip install -r requirements.txt
```

インストールされるライブラリ：

| ライブラリ | 用途 |
|-----------|------|
| pyautogui | スクリーンショット・キーボード操作の自動化 |
| pillow | 画像処理・PDF生成 |
| psutil | 実行中のプロセス（Kindleアプリ）の検出 |
| pywin32 | Windowsウィンドウの操作（最前面化・フルスクリーン化） |

## 使い方

### 1. Kindleアプリを起動してキャプチャしたい本を開く

あらかじめKindleアプリで本を開き、最初のページを表示した状態にしておいてください。

### 2. スクリプトを実行する

```bash
python kindle_capture.py
```

### 3. 設定ウィンドウに入力する

以下のウィンドウが表示されます。

| 項目 | 入力内容 | 例 |
|------|----------|----|
| 作者 | 本の著者名 | 山田太郎 |
| タイトル | 本のタイトル | Python入門 |
| ページ送り | 本のページをめくる方向 | 左送り（日本語本）/ 右送り（英語本） |
| ページ数 | キャプチャするページ数 | 200（空欄にすると自動検出） |

「開始」を押すと、自動的にKindleウィンドウに切り替わり、キャプチャが始まります。

### 4. キャプチャ中はPCを触らない

自動でページをめくりながら画像を保存します。
マウスやキーボードを操作すると誤動作する場合があります。

### 5. 完了後の後処理

キャプチャ完了後、ダイアログが表示されます。

- **「はい」** → 一時ファイル（PNG画像）を削除し、PDFを最終フォルダに移動
- **「いいえ」** → 一時ファイルをそのまま残す

## 出力ファイルの場所

```
output/
├── temp_20260330_120000/   # キャプチャ中の一時ファイル（PNG画像）
│   ├── 0001.png
│   ├── 0002.png
│   └── ...
└── final_pdf/              # 完成したPDF
    └── 山田太郎_Python入門.pdf
```

PDFのファイル名は `作者_タイトル.pdf` の形式で自動生成されます。

## よくあるトラブル

| 症状 | 原因と対処法 |
|------|-------------|
| 「Kindleのウィンドウが見つかりませんでした」 | KindleアプリをPC上で起動してから再度実行してください |
| ページが同じ画像ばかり保存される | ページ送り方向（左/右）が逆になっています |
| `pip install` でエラーが出る | Pythonのバージョンが古い可能性があります。3.10以上を使用してください |
| `(.venv)` が表示されない | 仮想環境が有効化されていません。`.venv\Scripts\activate` を実行してください |

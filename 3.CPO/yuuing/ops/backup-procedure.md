# WordPressバックアップ手順

## 推奨頻度
- **DBバックアップ**: 月1回（議事録・投稿追加時）
- **ファイルバックアップ**: テーマ・プラグイン変更時

## 方法1: Xserverの自動バックアップ（推奨）

Xserverにはサーバーパネルから過去14日分のバックアップをダウンロードできる機能がある。

1. https://www.xserver.ne.jp/login_server.php にログイン
2. 「WordPress簡単バックアップ」または「バックアップ」メニュー
3. `yuuing.co.jp` を選択してダウンロード

## 方法2: WordPress管理画面からエクスポート

1. 管理画面 → ツール → エクスポート
2. 「すべてのコンテンツ」を選択
3. XMLファイルをダウンロード → `wordpress/backups/YYYY-MM-DD_export.xml` として保存
